# Clonación de repositorio
*Git debe estar instalado en el equipo para poder clonar el repositorio*
## Pasos de clonación
1. Ingresamos la consola de git
2. Cambiamos de directorio con `cd` hasta la carpeta que queramos alojar el repositorio
3. ejecutamos el siguiente comando: `git clone https://github.com/LaYer61/pagina_ejemplo.github.io.git`
4. Revisamos que tengamos los mismos archivos que el repositorio, si queremos ingresar al repositorio, aqui esta el link: https://github.com/LaYer61/pagina_ejemplo.github.io.git
5. Si queremos ingresar a github page del repositorio, el link es: https://layer61.github.io/pagina_ejemplo.github.io/
